# עמוד תרומה - עמותת שערות בסערות

עמוד תרומה לעמותת **שערות בסערות** ([peot.org.il](https://www.peot.org.il/)) - פאות רפואיות לנשים המתמודדות עם סרטן, מהשיער הטבעי שלהן.

## 🔗 תצוגה חיה (ללקוח)
https://bzik23.github.io/peot-preview/
> `noindex` - לא מאונדקס בגוגל. תצוגה בלבד, טרם עלה לאוויר.

## קבצים
| קובץ | תיאור |
|------|--------|
| `index.html` | עמוד התצוגה העצמאי המוגש ב-GitHub Pages (עם פונט Leon + הדר/פוטר-לס). |
| `donate-wp-source.html` | **קוד המקור להטמעה ב-WordPress** - בלוק `wp:html` scoped תחת `.peot-donate`, נכנס לעמוד 696. |
| `peot-hero.mp4` | וידאו רקע ההירו (~1.6MB, מושתק, לופ 12.5ש'). |
| `peot-hero-poster.jpg` | תמונת poster לטעינה מהירה / reduced-motion. |

## ההירו (וידאו רקע)
- **מקור הווידאו:** הסרטון הרשמי של העמותה ביוטיוב (`YuLpmHif8VE`, 360p - האיכות היחידה שהועלתה).
- **הלופ:** קטע ~t66-78.5 - עיצוב פאת תלתלים ← חיוך ← סלפי חגיגי. מושתק, בלי כתוביות (נבחר קטע נקי + קיצוץ תחתון), הובהר בקידוד (gamma/brightness/saturation), unsharp.
- **עיצוב:** תמה בהירה תואמת האתר - רקע ורוד-פודרה (`#FDF4F8`/`#FBEAF2`), כותרות בורדו (`#36001C`), אקסנט מג'נטה (`#C41D72`), פונט Leon.
- **קומפוזיציה:** טקסט בשליש העליון מעל veil ורוד מדורג; פני האישה נחשפים בהיר מתחת (בלי חפיפה).
- **אלמנטים בשפת האתר:** כפתור פיל מג'נטה עם חץ, קישור-טקסט עם לב וקו תחתון, קווי SVG זורמים (מוטיב השיער).
- נגישות: poster + fallback ל-`prefers-reduced-motion`, התאמת מובייל.

## עדכון התצוגה
```
# מעדכנים index.html / נכסים, ואז:
git add -A && git commit -m "..." && git push
```
GitHub Pages בונה מחדש תוך דקה-שתיים (רענון קשה: Ctrl+F5).

## הטמעה ל-WordPress (עמוד 696, כרגע טיוטה)
1. להעלות את `peot-hero.mp4` + `peot-hero-poster.jpg` ל-Media Library (או להשאיר את כתובות ה-GitHub Pages המוחלטות שכבר בקוד).
2. להדביק את תוכן `donate-wp-source.html` לעמוד 696 (כבלוק `wp:html` - מבטל wpautop).
3. לנקות cache ולאמת אנונימית.
